For master:
- install FastAPI (follow https://fastapi.tiangolo.com/)
	pip install fastapi
- install uvicorn 
	pip install "uvicorn[standard]"
  
For SLAVE:
- install MQTT client (follow https://www.hivemq.com/blog/mqtt-client-library-paho-python/)
	pip install paho-mqtt
	or
	pip3 install paho-mqtt

- set GAME_WRAPPER_CONFIG_FILE env
